# Changelog

## [1.6.0] - 2026-01-25

### Key Features
- **Round & Oval Frames**: Added full support for non-rectangular frames. You can now toggle shapes in the Properties panel.
- **Advanced Filtering & Sorting**:
  - **Frame Library**: Fuzzy search ("8 x 10"), filter by "Placed/Unplaced", and sort by Size/Dimensions.
  - **Photo Library**: Search by filename (for new uploads), filter by "Used/Unused", and Smart Orientation filters.
- **Custom Labels**: Name your frames (e.g., "Hallway", "Grandma") for easier identification in the library.
- **Smart Matting**: Toggle matting on/off and adjust opening dimensions directly in the Properties panel.

### Improvements
- **Orientation Control**: "Rotate Frame" button to quickly swap width and height.
- **Image Management**: Improved crop/pan state management (resets handling when replacing images).
- **Frame Colors**: Added color presets (Black, White, Wood, Gold, Silver) and custom color picker.
- **Fuzzy Search**: "8 x 10" now correctly matches "8x10".

### Fixes
- Fixed sort order keying in Photo Library (Oldest/Newest).
- Fixed issues with frame dimensions not updating the library template.
