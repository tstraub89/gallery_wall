# Gallery Wall Layout Planner

A "vibe-coded" interactive web application for planning multiple gallery wall projects.

## Features

-   **Infinite Canvas**: Pan, zoom, and arrange frames freely.
-   **Frame Library**: Import text files (e.g., `staircase_sizes.txt`) or add frames manually.
-   **Visualizations**: See matting, toggle wall templates (staircase), and drag-and-drop real images.
-   **Project Management**: Manage multiple projects with auto-save to local storage.
-   **Export**: Backup projects to JSON or generate shopping lists.

## Getting Started

1.  `npm install`
2.  `npm run dev`

## Built With

-   React
-   Vite
-   Vanilla CSS (Modules)
